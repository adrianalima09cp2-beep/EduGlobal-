# EduGlobal 🌍📚

Conteúdos destinados a educação para diminuir a desigualdade na educação não apenas no Brasil, mas no mundo.

## Tópicos Principais

1. **Fundamentos da Educação Global**
   - Princípios e objetivos da educação inclusiva
   - Impacto da educação na redução de desigualdades
   - Histórico de iniciativas educacionais mundiais

2. **A verdade história do Brasil**
   - Contexto histórico da educação no Brasil
   - Evolução do sistema educacional brasileiro
   - Desafios e avanços na educação brasileira
   - Referências e estudos históricos

## Objetivos

- ✅ Democratizar o acesso à educação de qualidade
- ✅ Promover conteúdos educacionais acessíveis
- ✅ Reduzir a desigualdade educacional global
- ✅ Inspirar mudanças positivas através da educação

## Como Contribuir

Contribuições são bem-vindas! Para participar:

1. Faça um fork do projeto
2. Crie uma branch para sua feature (`git checkout -b feature/MinhaContribuicao`)
3. Commit suas mudanças (`git commit -m 'Adiciona contribuição'`)
4. Push para a branch (`git push origin feature/MinhaContribuicao`)
5. Abra um Pull Request
