Bem-vindo ao **EduGlobal**! Um projeto educacional destinado a alunos de **todos os níveis** - desde o ensino fundamental até o ensino superior e educação continuada.

## 📖 Sobre o Projeto

Conteúdos destinados a educação para diminuir a desigualdade na educação não apenas no Brasil, mas no mundo. Aqui você encontrará materiais de aprendizado, recursos educacionais e informações confiáveis para apoiar sua jornada educacional.

## 🎯 Para Quem?

Este projeto é criado **por alunos e para alunos** de todos os níveis:
- 🏫 Ensino Fundamental
- 📚 Ensino Médio
- 🎓 Ensino Superior
- 🌱 Educação de Jovens e Adultos
- 💡 Aprendizado Contínuo

## 📚 Tópicos Principais

### 1. **Fundamentos da Educação Global**
   - Princípios e objetivos da educação inclusiva
   - Impacto da educação na redução de desigualdades
   - Como aprender melhor e de forma significativa

### 2. **A verdade sobre a história do Brasil**
   - Descobrimento e colonização
   - Evolução do Brasil ao longo dos séculos
   - Personalidades históricas importantes
   - Eventos que marcaram a história brasileira
   - Desafios e conquistas da educação no Brasil
   - **Para todos os níveis**: desde conceitos básicos até análises aprofundadas

## 🚀 Primeiros Passos

### Como baixar e estudar um conteúdo

#### Pré-requisitos
- Git instalado em seu computador
- Navegador web
- Um editor de texto (VS Code, Notepad++, etc.)

#### Instalação

1. **Clone o repositório**
   ```bash
   git clone https://github.com/adrianalima09cp2-beep/EduGlobal-.git
   cd EduGlobal-
   ```

2. **Explore os conteúdos**
   ```bash
   # Veja as pastas disponíveis
   ls -la
   ```

3. **Escolha um tópico e comece a estudar**
   ```bash
   # Exemplo: acessar conteúdo de História do Brasil
   cd historia-do-brasil
   cat README.md
   ```

4. **Abra os arquivos em seu editor favorito**
   - Os conteúdos estão em formato Markdown (.md)
   - Você pode ler direto no GitHub ou fazer download e estudar offline

#### Exemplo Rápido
```bash
# 1. Clone
git clone https://github.com/adrianalima09cp2-beep/EduGlobal-.git

# 2. Entre na pasta
cd EduGlobal-

# 3. Leia o conteúdo
cat historia-do-brasil/README.md

# 4. Faça os exercícios propostos e aprenda!
```

## 💪 Como Contribuir

Alunos, professores e educadores podem contribuir! 

Veja o guia completo em [CONTRIBUTING.md](CONTRIBUTING.md) para mais detalhes.

**Passos rápidos:**
1. Faça um fork do projeto
2. Crie uma branch para sua contribuição (`git checkout -b feature/MinhaContribuicao`)
3. Adicione seus conteúdos, exercícios ou correções
4. Commit suas mudanças (`git commit -m 'docs: add novo conteúdo'`)
5. Push para a branch (`git push origin feature/MinhaContribuicao`)
6. Abra um Pull Request
