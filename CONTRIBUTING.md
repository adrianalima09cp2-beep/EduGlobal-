# Guia de Contribuição - EduGlobal

Obrigado por se interessar em contribuir para o **EduGlobal**! 🌍

Este projeto tem como missão criar conteúdos educacionais de qualidade para reduzir a desigualdade na educação, não apenas no Brasil, mas no mundo todo.

## Como Contribuir

### 1. **Reportar Problemas (Issues)**
- Encontrou um erro ou informação desatualizada?
- Abra uma [issue](../../issues) descrevendo o problema
- Forneça contexto e, se possível, sugestões de melhoria

### 2. **Adicionar Novo Conteúdo**
- Crie pastas organizadas por tema (ex: `historia-do-brasil/`, `ciencias/`)
- Estruture com `README.md` como índice
- Use linguagem clara e acessível
- Adicione referências e fontes
- Inclua imagens ou diagramas quando apropriado

### 3. **Melhorar Conteúdo Existente**
- Corrija erros gramaticais ou factuais
- Expanda seções incompletas
- Atualize informações desatualizada
- Melhore a formatação e legibilidade

### 4. **Enviar Pull Requests**
1. Faça um Fork do projeto
2. Crie uma branch: `git checkout -b feature/seu-conteudo`
3. Commit suas mudanças: `git commit -m "docs: add XYZ"`
4. Push para a branch: `git push origin feature/seu-conteudo`
5. Abra um Pull Request descrevendo sua contribuição

## Padrões de Contribuição

### Formatação de Arquivos
- Use **Markdown** para todos os documentos
- Estruture com headers claros (`#`, `##`, `###`)
- Mantenha linhas com até 80 caracteres quando possível
- Use listas e tabelas para organizar informações

### Commits
- Use mensagens descritivas: `docs: add content about X`
- Seja específico: `fix: correct date in history`, não `update`
- Referencie issues quando relevante: `Fixes #123`

### Conteúdo
- Verifique ortografia e gramática
- Use fontes confiáveis e citar quando necessário
- Mantenha tom educativo e inclusivo
- Evite linguagem complexa desnecessária

## Áreas Prioritárias

Estamos buscando contribuições em:

- ✅ **História**: Expandir cronologia brasileira e mundial
- ✅ **Ciências**: Conceitos fundamentais e práticas
- ✅ **Matemática**: Tutoriais e resolução de problemas
- ✅ **Línguas**: Dicas de aprendizado
- ✅ **Educação Inclusiva**: Conteúdos em formatos acessíveis

## Código de Conduta

Todos os contribuidores devem:
- Ser respeitosos e inclusivos
- Aceitar críticas construtivas
- Focar no melhor interesse do projeto educacional
- Reportar comportamentos inapropriados

## Dúvidas?

- Abra uma [Discussion](../../discussions) para perguntas
- Entre em contato através das issues
- Consulte a [README](../README.md) principal

Obrigado por ajudar a democratizar a educação! 🎓✨
